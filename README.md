# 🎮 Jogo da Forca

Projeto desenvolvido em Python com o objetivo de criar uma versão interativa do clássico jogo da forca no terminal, com escolha de temas e palavras aleatórias.

## 📌 Sobre o Projeto

Este jogo da forca permite ao jogador escolher entre 5 temas diferentes e tentar adivinhar uma palavra sorteada aleatoriamente dentro do tema. O jogador pode digitar letras ou tentar adivinhar a palavra completa. A cada erro, uma parte do boneco da forca é desenhada.
